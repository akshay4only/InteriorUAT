# Bespoke Interior Design Portfolio

A luxury interior design portfolio built with Next.js 15, Tailwind CSS, and Framer Motion.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```

## Project structure

- `app/` - Next.js App Router pages and layout
- `components/ui/` - reusable UI components
- `lib/animations.ts` - centralized motion variants
- `tailwind.config.ts` - design tokens and responsive utilities
